✓ APPLICATION FEATURES

  Dashboard (/)
    → Summary stats (active customers, scans, critical risks)
    → Recent activity feed
    → Monthly assessment trend chart

  Customers (/customers)
    → View all customers in your tenant
    → Create new customer engagements
    → Risk score indicators

  Customer Workspace (/customers/[id])
    → Overview: Risk gauge, PII distribution, readiness summary
    → Repositories: Connect data sources (S3, OneDrive, SharePoint)
    → Scans: Run and monitor DPDP scans with real-time status
    → Inventory: Search, filter, paginate PII findings
    → Reports: Generate and download assessment reports

  Settings (/settings)
    → View account information
    → Configure preferences
    → API configuration (future)
